# Quick start guide

This guide will help you tile your first window with MouseTiler using the Grid Tiler, the simplest and most intuitive tiling mode.

<Note>
  Make sure you've already [installed MouseTiler](/installation) before proceeding.
</Note>

## Tile your first window with Grid Tiler

The Grid Tiler is the fastest way to start tiling windows. It activates automatically when you move a window.

<Steps>
  <Step title="Grab a window">
    Click and hold the title bar of any window to start dragging it.
  </Step>

  <Step title="Move the window slightly">
    Move the window just a few pixels. The Grid Tiler popup will appear automatically.
    
    <img src="/assets/popup_tiler.png" alt="Grid Tiler popup" />
  </Step>

  <Step title="Hover over a tile">
    Move your mouse over any tile in the grid. The tile will highlight to show where your window will be placed.
  </Step>

  <Step title="Release to tile">
    Release the mouse button to place your window in the selected tile. Your window will automatically resize and position itself!
  </Step>
</Steps>

That's it! You've just tiled your first window with MouseTiler.

## Try different layouts

The Grid Tiler comes with multiple predefined layouts. To switch between them:

1. While dragging a window, move your mouse to the **top row** of the Grid Tiler popup
2. Each option in the top row represents a different layout
3. Hover over a layout to preview it, then select a tile below

<img src="/assets/popup_tiler_all.png" alt="Multiple Grid Tiler layouts" />

Common layouts include:
- **1x1**: Full screen
- **2x1**: Left and right halves
- **3x1**: Three vertical columns
- **2x2**: Four equal quadrants
- **Custom layouts**: Many specialized layouts for different workflows

## Essential keyboard shortcuts

While MouseTiler is designed for mouse-based tiling, keyboard shortcuts can enhance your workflow:

### Toggle input mode

```
Ctrl+Alt+I
```

Switch between mouse cursor mode and window-based input. Useful for different input devices like stylus or touch.

### Center in tile

```
Meta+Ctrl+C
```

Keep your window's current size but center it within a tile. Perfect for windows that don't need to fill the entire tile.

<img src="/assets/center_in_tile_v1.gif" alt="Center in tile demo" />

### Toggle tiling suggestions

```
Meta+Ctrl+S
```

Show or hide window tiling suggestions.
