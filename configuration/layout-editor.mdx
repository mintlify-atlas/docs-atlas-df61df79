MouseTiler includes a powerful web-based layout editor that makes it easy to visually design custom tiling layouts without writing layout strings manually.

## Accessing the layout editor

The layout editor is available at:

**https://rxweb.epizy.com/mousetiler/editor.html**

You can access this editor from any web browser - no installation required.

## Using the layout editor

### Creating a new layout

<Steps>
  <Step title="Open the editor">
    Navigate to the [MouseTiler Layout Editor](https://rxweb.epizy.com/mousetiler/editor.html) in your web browser
  </Step>
  
  <Step title="Design your layout">
    Use the visual tools to create tiles by:
    - Clicking and dragging to create new tiles
    - Adjusting tile sizes and positions
    - Adding multiple tiles to create complex layouts
  </Step>
  
  <Step title="Preview the layout">
    See a real-time preview of how your layout will appear in MouseTiler
  </Step>
  
  <Step title="Export the layout string">
    Once satisfied with your design, export the layout as a text string
  </Step>
</Steps>

### Layout format

The editor generates layout strings in MouseTiler's format:

**Grid format:**
```
XxY
```
Example: `4x2` creates a 4-column by 2-row grid

**Tile format:**
```
X,Y,Width,Height
```
- Values can be in **percent** (e.g., `50` or `33.33`)
- Values can be in **fractions** (e.g., `1/3`)
- Values can be in **pixels** with `px` suffix (e.g., `100px`)

Example: `0,0,50,100` creates a tile at the left side, half the screen width, full height

**Anchored tile format:**
```
X,Y,Width,Height,AnchorX,AnchorY
```
Anchors: `LEFT`, `CENTER`, `RIGHT` (horizontal) and `TOP`, `CENTER`, `BOTTOM` (vertical)

Example: `100,100,50,33,RIGHT,BOTTOM` creates a tile in the bottom-right corner

**Multiple tiles:**
```
tile1+tile2+tile3
```
Example: `0,0,50,100+50,0,50,100` creates two side-by-side tiles (same as `2x1`)

## Importing layouts into MouseTiler

### For Overlay mode

<Steps>
  <Step title="Copy layout string">
    Copy the exported layout string from the editor
  </Step>
  
  <Step title="Open MouseTiler settings">
    Go to **System Settings** > **Window Management** > **KWin Scripts** > **Mouse Tiler** (Configure)
  </Step>
  
  <Step title="Navigate to Overlay tab">
    Click on the **Overlay** tab in the settings
  </Step>
  
  <Step title="Paste layout string">
    Paste your layout string into the **Layout** field
  </Step>
  
  <Step title="Apply and reload">
    Click **OK**, then reload the script (see [Settings](/configuration/settings) for reload instructions)
  </Step>
</Steps>

### For Popup Grid mode

<Steps>
  <Step title="Copy layout string">
    Copy the exported layout string from the editor
  </Step>
  
  <Step title="Open MouseTiler settings">
    Go to **System Settings** > **Window Management** > **KWin Scripts** > **Mouse Tiler** (Configure)
  </Step>
  
  <Step title="Navigate to Popup Grid Layouts tab">
    Find the layouts configuration section
  </Step>
  
  <Step title="Add layout string">
    Add your layout string to either:
    - **Popup Grid Layouts** - For frequently used layouts
    - **All Layouts** - For additional layouts accessible via Ctrl+Space
  </Step>
  
  <Step title="Apply and reload">
    Click **OK**, then reload the script (see [Settings](/configuration/settings) for reload instructions)
  </Step>
</Steps>

## Exporting and sharing layouts

### Export to file

The layout editor allows you to:

1. **Save layouts** - Export your layout configurations to a file for backup
2. **Share layouts** - Share layout strings with other MouseTiler users
3. **Import layouts** - Load previously saved layout configurations

### Layout collections

You can create multiple layouts and organize them:

- **Single layout** - One layout string for Overlay mode
- **Layout list** - Multiple layouts separated by newlines for Popup Grid mode
- **Named layouts** - Add names using semicolon: `layout;Name`

Example of multiple named layouts:
```
2x1;Two columns
1x2;Two rows
0,0,67,100+33,0,67,100+17,0,66,100;67 66 67 (%)
```

## Advanced layout features

### Using fractions

For precise divisions, use fractions instead of decimals:

```
0,0,1/3,100+1/3,0,1/3,100+2/3,0,1/3,100
```

This creates three equal columns with perfect precision.

### Using pixels

For fixed-size tiles, use pixel values:

```
0px,0px,500px,300px
```

This creates a 500x300 pixel tile in the top-left corner.

### Combining formats

You can mix percentages, fractions, and pixels:

```
0,0,50,100+50px,0,200px,100
```

Left tile takes 50% width, right tile is fixed 200px wide starting at 50px from left edge.

## Tips for layout design

<Note>
  **Start simple** - Begin with basic grid layouts (2x1, 2x2) before creating complex custom layouts
</Note>

<Note>
  **Test layouts** - After importing a layout, test it by moving windows to ensure tiles align as expected
</Note>

<Note>
  **Use the visual editor** - The web editor is much easier than manually writing layout strings
</Note>

<Warning>
  **SPECIAL layouts not supported in Overlay** - Special layouts like FILL, SPLIT, MAXIMIZE only work in Popup Grid mode
</Warning>

## Common layout examples

### Two-thirds / one-third split
```
0,0,67,100+67,0,33,100
```

### Three equal columns
```
3x1
```
or
```
0,0,1/3,100+1/3,0,1/3,100+2/3,0,1/3,100
```

### Main window with sidebar
```
0,0,75,100+75,0,25,100
```

### Centered window
```
25,25,50,50
```

### Complex 6-tile layout
```
0,0,50,100+50,0,25,50+50,50,25,50+75,0,25,50+75,50,25,50
```

Use the [web editor](https://rxweb.epizy.com/mousetiler/editor.html) to visualize and create these layouts!
