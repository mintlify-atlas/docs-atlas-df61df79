## Current version

### Version 6.1.1

The latest stable release of MouseTiler. This version includes all core features:

- Two mouse tiling modes (Popup Grid and Overlay)
- Virtual desktop management
- Auto tiling (carousel and stacking styles)
- Manual and web-based layout configuration
- Multi-monitor support
- System theme integration and custom color themes
- Over 20 customization settings
- Support for mouse, stylus, and touch input (including Wacom)

<Note>
  MouseTiler uses semantic versioning (MAJOR.MINOR.PATCH). Version 6.1.1 indicates KDE Plasma 6 compatibility with incremental feature updates and bug fixes.
</Note>

## Development model

### Donation-based features

As of February 2026, new features are developed based on community donations. This model ensures sustainable development while keeping the extension free to use.

<Info>
  **Bug fixes and KDE compatibility updates remain free for everyone.** Only new feature development is donation-based.
</Info>

#### How it works

<Steps>
  <Step title="Vote on features">
    Community members vote on which features they want next through the [WakaTime poll](https://wakatime.com/polls/0e8054f0-e168-4f00-b31b-ed6c17bd51af)
  </Step>
  <Step title="Contribute via donations">
    Donations accumulate toward the top-voted feature's goal
  </Step>
  <Step title="Feature implementation">
    Once the donation goal is reached, the developer implements that feature
  </Step>
  <Step title="Release to everyone">
    The new feature is released for free to all MouseTiler users
  </Step>
</Steps>

## Planned features

The community is voting on numerous planned features. Here are some of the top-voted items:

<AccordionGroup>
  <Accordion title="Keyboard tiling shortcuts (M - 1 day)">
    Add keyboard shortcuts for tiling and moving already-tiled windows. For example, `Ctrl+Meta+Num 4` to move a window left in the current layout. Includes directional movement (left, right, up, down) with customizable keybindings.
  </Accordion>

  <Accordion title="Custom size split tile (S - Half day)">
    Make it possible to use custom sizes for split tiles (SPECIAL_SPLIT_HORIZONTAL and SPECIAL_SPLIT_VERTICAL), such as 70%/30% instead of the default 50%/50%.
  </Accordion>

  <Accordion title="Portrait mode display setting (S - Half day)">
    Add a setting to invert width/height on portrait monitors, optimizing the tiler display for vertical orientations.
  </Accordion>

  <Accordion title="Per tile modifiers (M - 1 day)">
    Add per-tile modifiers so you can both tile and perform special operations at once. Examples include: center in tile, fullscreen, keep above, keep below, or close. Visual hints would show what each tile does (C for center, F for fullscreen, etc.).
  </Accordion>

  <Accordion title="Per screen layout configuration (L - 2 days)">
    For advanced users who want different tiling options per screen in multi-monitor setups.
  </Accordion>

  <Accordion title="Snap adjacent tiles / resize together (XL - 3-5 days)">
    Resize adjacent windows together, add boundaries so you resize until reaching a new boundary, and resize windows below if edges match the currently resized edge.
  </Accordion>

  <Accordion title="Maximize/Minimize restore action (L - 2 days)">
    Decide where windows go after being manually maximized or minimized and then restored. Options include inserting at exact index, scrolling to restored window, or inserting before/after specific windows.
  </Accordion>

  <Accordion title="Multi-layout web editor (XL - 3-5 days)">
    Edit multiple layouts at once in the web editor. Copy your entire layout list from settings, rearrange and edit layouts one by one, then export them back to settings.
  </Accordion>
</AccordionGroup>

## Version checking

To check your installed version of MouseTiler:

<Steps>
  <Step title="Open KWin Scripts">
    Navigate to System Settings > Window Management > KWin Scripts
  </Step>
  <Step title="Find MouseTiler">
    Look for Mouse Tiler in the list of installed scripts
  </Step>
  <Step title="View version">
    The version number is displayed next to the extension name
  </Step>
</Steps>

Alternatively, you can check the metadata file:

```bash
# Find MouseTiler installation
find ~/.local/share/kwin/scripts -name "metadata.json" -path "*/mousetiler/*"

# View version in metadata
cat ~/.local/share/kwin/scripts/mousetiler/metadata.json | grep Version
```
